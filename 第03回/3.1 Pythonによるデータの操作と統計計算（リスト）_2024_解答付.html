3.1 Pythonによるデータの操作と統計計算（リスト）

データをまとめて管理する方法¶

  • 複数の値を1つの変数で管理しながら、個々の値（要素と呼ぶ）を取り出したり、場合によっては変更したりできると便利なことがある。そのためにPythonではリスト、タプル、辞書、集合のデータ型が提供されている
  • 本講義では、特に重要な、リストを学習する

リストの書き方¶

  • リストは、複数の値をまとめて扱う場合に使用
  • リストは全体を[]で囲み、各要素はカンマで区切る
  • リストではインデックス(要素番号)を用いて、要素の追加や入れ替えなどが可能
  • リストは型の値でも要素として格納することができ、リストの中にリストを格納することも可能である
  • リストは、数値や文字列などの基本データ型と並んでPythonでは高頻度で使われるデータ型

複数のリストの結合¶

リストA = リストB + リストC

listA = [90, 85, 75, 95, 80]

# リストを出力
print("2回目の結合：", listD)

1回目の結合： [90, 85, 75, 95, 80, 70, 80, 75, 90]
2回目の結合： [90, 85, 75, 95, 80, 70, 80, 75, 90, 85, 95]

リストの要素の並び替え¶

  • sorted関数を用いる
    • 昇順に並び替え（小→大）
      sorted(リスト)
    • 降順に並び替え（大→小）
      sorted(リスト,reverse=True)`

要素が数値の場合¶

In [23]:
listD = [90, 85, 75, 95, 80, 70, 80, 75, 90, 85]

# listDを昇順に並び替えてlistEに代入

print("昇順に並び替え: ", listE)

降順に並び替え:  [95, 90, 90, 85, 85, 80, 80, 75, 75, 70]

リスト内の要素の統計計算¶

  • 標準ライブラリのstatisticsを用いて、平均、中央値、最頻値、分散、標準偏差を算出できる
    • ライブラリとは、よく使う機能・関数をまとめて、簡単に使えるようにしたもの
    • ライブラリの中身のプログラムがブラックボックスであっても、入力と出力がわかっていれば簡単に実装し、実行することができる
    • Pythonに便利な機能を持つライブラリが多数ある
  • 標準ライブラリを使う場合、import文で呼び出す
    import statistics
statistics.mean(リスト)      # 平均値
statistics.median(リスト)    # 中央値(要素を小さい方から大きい方に並べたときの中央の値)
statistics.mode(リスト)      # 最頻値(最も出現頻度が高い値)
statistics.pvariance(リスト) # 分散(データのばらつき)
statistics.pstdev(リスト)    # 標準偏差(分散の平方根)

10人のテストの点数の統計量を計算

In [30]:
import statistics

# 10人の点数
listJ = [90, 85, 75, 95, 80, 70, 75, 90, 80, 75] 

mean = statistics.mean(listJ)
print("平均値: ",  mean)

median = statistics.median(listJ)
print("中央値: ", median)

mode = statistics.mode(listJ)
print("最頻値: ", mode)

pvariance = statistics.pvariance(listJ)
print("分散: ", pvariance) 

pstdev = statistics.pstdev(listJ)
print("標準偏差: ", pstdev)

平均値:  81.5
中央値:  80.0
最頻値:  75
分散:  60.25
標準偏差:  7.762087348130012
